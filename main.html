Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select an element in the DOM.
1.document.getElementByTagName
2.document.getElementById
3.document.querySelector
4.document.querySelectorAll
5.document.getElementsByClassName

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML get or set a block of html inside of a element, innerText get or set the text inside of a tag, innerContent

Question-3: What is event delegation in the DOM?

Event delegation is a method where a parent element handle events for its child elements. Instead of adding event listeners to every child, just applied once to the parent

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event on a child element "bubbles up" through its parent elements in the DOM tree

Question-5: How do you create, add, and remove elements using JavaScript?

For Creating document.createElement();
For Adding document.body.appendChild();
For Removing document.body.removeChild();
